## GET

### Available Options:

* ORDER_BY
    
    Accepts a field name that is present in the database.

* ORDER

     Optional if ORDER_BY is set. Defaults to ascending. 
     Available options: "ASC" , "DESC"

* LIMIT

    Accepts an integer to limit amount of responses to. Defaults to 50
